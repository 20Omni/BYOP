# Lane Detection and Driver Assistance System

## Overview

This project is a simple lane detection system built using Python and OpenCV.  
It can detect road lanes from images, videos, and live camera input.

The main goal is to identify lane markings and give basic feedback about vehicle position.

---

## Problem

While driving, it is important to stay within lane boundaries.  
Many vehicles do not have built-in systems for this.

This project tries to solve that problem using basic computer vision techniques.

---

## Features

- Works on:
  - Image input
  - Video input
  - Live camera
- Detects lane lines
- Highlights lane region
- Shows simple driving feedback

---

## Technologies Used

- Python  
- OpenCV  
- NumPy  
- Streamlit  

---

## How to Run

### Step 1: Install dependencies

pip install -r requirements.txt

### Step 2: Run the project

streamlit run app.py

### Step 3: Open in browser

http://localhost:8501
